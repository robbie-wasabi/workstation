## Initial Setup

```sh 

# install brew
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# install iterm2
$ brew install iterm2

# neovim
$ brew install neovim
$ cp -R ./neovim ~/.config/

```
